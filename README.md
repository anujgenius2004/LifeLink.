# LifeLink.
**Decentralized Offline Emergency Response System using Sensor Fusion and SMS Fallback**.

LifeLink is an Android-based safety application designed to detect vehicle accidents using sensor fusion and trigger emergency responses in network-blind zones.

## 🚀 Try it Now!
You don't need to install anything on your PC. You can test the live app on your phone:

1. Download the **Expo Go** app from the Play Store/App Store.
2. **Click here to open the Project on Expo Snack-->** (https://snack.expo.dev/@anuj2004/lifelink)
3. Scan the QR code displayed on the webpage using the Expo Go app.

## 🛠 Features
- **Automatic Crash Detection:** Uses Accelerometer G-Force thresholding (>4.0G).
- **Offline SMS Beacon:** Automatically generates a Google Maps link for emergency contacts.
- **Fail-Safe Mechanism:** 10-second countdown to prevent false alarms.
- **High-Contrast UI:** Designed for visibility in emergency situations.

## 🏗 Tech Stack
- **Framework:** React Native (Expo)
- **Sensors:** Accelerometer (expo-sensors)
- **Location:** Geolocation API (expo-location)
- **Messaging:** SMS API (expo-sms)
